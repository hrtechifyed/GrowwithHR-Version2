# Repository Update Notes

This repository is a working copy of GrowWithHR with recent updates for the executive assessment UX layer.

## Latest Update

- Added a React/Next.js-ready UX layer under `apps/web/src`.
- Added animated assessment intro, scene transitions, wizard flow, contact capture, privacy/data-handling panels, and a personalized report generator.
- Updated documentation to match the current no-storage policy.

## Current Data Policy

Assessment data is held only in the open browser tab. It is not stored in localStorage, sessionStorage, cookies, backend services, Google Drive, CRM, email tools, or HRTechify admin dashboards.

If the user leaves midway, refreshes, closes the tab, or switches devices, they must start fresh.
