# GrowWithHR v0.15.1-beta — M0 Baseline Release Manifest

## Release identity

- Product: GrowWithHR
- Milestone: M0 — Baseline Freeze
- Version: 0.15.1-beta
- Working branch: `feature/dna-m0-baseline`
- Stable production branch: `main`
- Recovery branch: `archive-v0.15.0-pre-dna`
- Release type: Baseline hardening
- Feature scope: No new Compliance DNA user experience

## Purpose

This release freezes and documents the existing GrowWithHR experience before the Compliance DNA transformation begins.

The release establishes:

- a recoverable pre-transformation baseline;
- one canonical application version;
- automated regression checks;
- stable assessment, persistence, PDF and email contracts;
- a controlled foundation for parallel-route development.

## Stable user routes

The following routes are part of the protected baseline:

- `/index.html`
- `/analyze-company.html`
- `/executive-advisory-report.html`
- `/sample-advisory-report.html`
- `/official-resources.html`
- `/more-info.html`

The existing `/analyze-company.html` route must remain operational during later milestones until an explicitly approved migration or retirement decision is recorded.

## Protected assessment flow

The stable assessment journey contains:

1. Landing screen
2. Guided organisation-story assessment
3. Organisation-story review
4. Contact and consent collection
5. Advisory generation
6. Completion, PDF download and email delivery

The baseline retains:

- four assessment chapters;
- seven story moments;
- start, resume and start-again actions;
- backward and forward navigation;
- editable review summaries;
- separate optional marketing consent.
