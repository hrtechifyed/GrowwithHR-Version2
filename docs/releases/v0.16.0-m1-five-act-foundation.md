# GrowWithHR v0.16.0-beta — M1 Five-Act Story Foundation

## Release identity

- Product: GrowWithHR
- Milestone: M1 — Five-Act Story Foundation
- Target version: `0.16.0-beta`
- Current integration version: `0.15.1-beta`
- Release type: Private-beta foundation
- Stable assessment route: `/analyze-company.html`
- Private-beta route: `/analyze-company-v3.html`
- Public feature flag default: disabled

## Purpose

M1 establishes an isolated Compliance DNA Five-Act presentation foundation without replacing or changing the existing Executive Advisory Briefing.

The five acts are:

1. Discover
2. People
3. Footprint
4. Understand
5. Act

This milestone does not introduce new compliance conclusions, recommendation rules or persisted answer meanings.

## Stable-route guarantee

The protected production assessment remains `/analyze-company.html`.

The stable route continues to own:

- assessment questions and validation;
- saved progress;
- recommendation logic;
- report generation;
- PDF generation;
- email delivery;
- contact and consent handling.

The private-beta route must not replace the stable route while `complianceDnaV3` remains disabled.

## Private-beta route

The isolated review route is `/analyze-company-v3.html`.

It provides:

- Five-Act navigation;
- accessible progress reporting;
- configurable narrative copy;
- stable-route fallback;
- private-beta diagnostics;
- `noindex, nofollow` protection;
- reduced-motion support.

Direct private review is permitted. Public promotion remains controlled through application configuration.

## Application configuration

The configuration file is `js/config/app-config.js`.

It defines:

- `assessmentV2: "analyze-company.html"`
- `assessmentV3: "analyze-company-v3.html"`
- `complianceDnaV3: false`

The default flag must remain `false` until an explicitly approved release enables the route.

The move to `0.16.0-beta` must be a coordinated release operation across every version-bearing file required by the version-sync contract.

The version must not be changed in only one file.

## Out of scope

M1 does not include:

- replacing the stable assessment route;
- publicly enabling the feature flag;
- adding new compliance questions;
- changing recommendation logic;
- changing the saved-state schema;
- redesigning report output;
- changing PDF or email contracts;
- retiring the v2 assessment.

## Rollback

M1 remains disabled publicly while `complianceDnaV3` is `false`.

The stable route remains `/analyze-company.html`.

No persistence migration is required for rollback because M1 does not replace existing storage contracts.
