# GrowWithHR v0.19 — M4 report integration and assessment fixes

## Purpose

This release connects the M4 explainability foundation to the executive PDF and delivery path used by the production advisory experience. It preserves the existing report and adds an auditable executive compliance section after the existing advisory pages.

## Visible report changes

The generated light and dark PDFs now include:

- an executive compliance summary with applicable, review-required and missing-information counts;
- separate organisation-profile, recommendation-confidence and evidence-availability indicators without a blended compliance score;
- a clear list of confirmed and missing inputs;
- prioritised statutory actions;
- a professional card for every governed law showing the threshold, reported organisation position, threshold result, why the law is included, required action, evidence coverage and missing information;
- direct official legislation links where an exact source is available;
- upcoming workforce-trigger warnings, including the maternity crèche threshold;
- an evidence-used section, recommendations roadmap and alphabetical governed-law appendix;
- the existing closing page after the new M4 section.

The prior 61–90 day roadmap decoration is also normalised so the stage label is no longer displayed as a mismatched box inside the larger roadmap card.

## Assessment fixes

- “Mainly on-site” automatically selects **None** for remote workforce and disables incompatible remote-percentage choices.
- “Mainly remote” automatically selects **Fully remote** and disables incompatible choices.
- Selecting two or three operating states requires the same number of different state names; missing or repeated states produce explicit guidance.
- The semicolon implementation note and visible separators are removed.
- Progress consistently uses **Step X of 7**, while the side rail remains a four-section overview.
- **Select all** appears directly below the guidance question and checks/highlights every priority.
- The PDF report-style selector is rendered as a contained fieldset.

## Email delivery

Light and dark PDFs are delivered in one customer email when both versions are selected. The server accepts one unified delivery request, attaches both PDFs, and creates one ZIP package when the combined raw attachments exceed the Gmail-safe threshold. One internal HRTechify notification records which version or versions were delivered. The founder name in the customer signature is plain text rather than bold.

## Safety

- No blended compliance, legal-confidence or maturity percentage is introduced.
- Evidence indicators represent confirmed assessment inputs only.
- The report remains general guidance and explicitly requires confirmation against current official sources and qualified advice.
- State Shops and Establishments links are not replaced with generic ministry landing pages; the report flags when an exact state source still needs confirmation.
